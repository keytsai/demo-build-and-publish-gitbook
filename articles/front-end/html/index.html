HTML

網頁欄位架構、表格、form表單等的快速建立方式範例(Emmet套件功能)

  • form>fieldset>legend{contact US}+(p>label+input)*6
  • header>h1+small^nav>a*4^video+#left>(h2+p*3)*4^#right>img*3+form^footer
  • div>h1{NEWS}^div+div
